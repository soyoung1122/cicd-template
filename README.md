# CI/CD 템플릿

## 소개

쿠버네티스 클러스터에 자체 CI/CD 파이프라인을 구축하는 방법에 대한 가이드 및 활용 가이드를 다루고 있다. 

-> 본 가이드에서는 CI/CD 파이프라인을 구축하고, 예시 애플리케이션을 배포하는 과정을 다루고 있다.

##### ※ CI/CD 서비스를 자체 구축하고자 하는 경우, 공모전 사무국에 노드 스펙 업그레이드를 요청한다.
